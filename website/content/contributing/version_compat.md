# Version Compatibility

The currently available versions of HARP and PyHARP are mutually compatible. Below, we provide compatibility information for all versions of HARP and PyHARP released to date:

| HARP | PyHARP |
| :-: | :-: |
| 2.2.0 | 0.2.1 |
| 2.1.x | 0.2.0 |
| 2.0.0 | 0.2.0 |
| 1.3.0 | 0.1.1 |
| 1.2.0 | 0.1.0 |
